# 🤖 AI Gesture Controlled Virtual Keyboard & Mouse

A real-time **AI Gesture Controlled Virtual Keyboard & Mouse** built using **Python, OpenCV, MediaPipe, and PyAutoGUI**. This project enables users to type and control the mouse using hand gestures captured through a webcam, providing a touch-free human-computer interaction experience.

---

## 🚀 Features

### ⌨️ Virtual Keyboard

* Gesture-based typing using pinch detection
* Hover key highlighting
* Caps Lock support
* Shift key support
* Space, Backspace, Enter, and Tab support
* Function key handling

### 🤖 AI Word Prediction

* Real-time word suggestions
* Clickable prediction boxes
* Smart word replacement
* Faster typing experience

### 🖱️ Virtual Mouse

* Cursor movement using index finger
* Left click
* Right click
* Drag and drop
* Scroll functionality
* Keyboard/Mouse mode switching

### 🎨 User Interface

* Modern dark theme
* Live FPS display
* Gesture detection display
* Pinch distance indicator
* Typing sound feedback

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/AI-Gesture-Controlled-Virtual-Keyboard-and-Mouse.git
```

### 2. Navigate to the project folder


```bash

```

## 🎮 Controls

### Keyboard Mode

* Move your index finger to hover over keys.
* Pinch your thumb and index finger to type.
* Select AI word predictions by hovering and pinching.

### Mouse Mode

* Press **M** to switch to Mouse Mode.
* Move your index finger to control the cursor.
* Pinch to left-click and drag.
* Two-finger gesture for right-click.
* Four-finger gesture for scrolling.
